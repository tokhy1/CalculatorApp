# Calculator Mobile App
Simple Calculator App using Jetpack Compose!


## Features

- Simple User interface
- Adding numners
- Substracting numbers
- Multipling numbers
- Dividing numbers
- Can make backspace (DEL)
- Can clear the result (AC)
